VX Software Upgrade Plan
This VX system has a 1-year Software Upgrade Plan (SUP) license.

Additionally, there is a "channel-year" balance on the license. One channel-year represents the use of one licensed channel for one year.
If the channel-year balance is consumed before the 1-year expiration date, the license will become invaild.
